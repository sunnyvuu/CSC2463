    • P5.js and Arduino Interaction Explained

    This web application demonstrates a dynamic interaction between a web-based interface and a physical computing device using P5.js and Arduino.

    Functionality Overview

    • Joystick Control: The X and Y positions of a joystick are read by the Arduino, which sends these positions to the sketch running in this web page. The positions are used to dynamically change the background color of the page:
      • Moving the joystick up turns the background green.
      • Moving it down turns the background red.
      • Moving it left turns the background blue.
      • Moving it right turns the background yellow.
      • Positioning it in the center resets the background to white.
    • LED Control: Pressing the 'R' and 'B' keys on the keyboard toggles red and blue LEDs, respectively, connected to the Arduino. This feature allows users to interact with physical hardware through digital input from the keyboard.
    • Serial Communication: The sketch uses serial communication to transmit joystick data from the Arduino to the web application and send control commands from the web application back to the Arduino to manage LED states.
    • Dynamic Serial Port Selection: Users can update the serial port through an HTML input field, allowing flexibility depending on which port the Arduino is connected to.

    Ensure the p5.serialcontrol application is running and configured correctly to facilitate communication between the Arduino and this web page.
